A digest of basic statistics from reading John Taylor's An Introduction to Error Analysis: The Study of Uncertainties in Physical Measurements

The book generally works "backwards": Starting from useful principles of working with uncertain values and then working through the justification and proofs. Since this is a quick digest and personal referesher/reference, the order is opposite of that presented in the book. When it makes sense I had inserted some MATLAB/Octave code to demonstrate things

Probability density functions (PDFs)

Probability density functions (PDFs) are the basic functions that describe a random variable. The basic property of a PDF is that it can be integrated between two values to get a probability that your random variable will generate a value between those two integration bounds. B/c the random variable will have some value, if you integrate between -∞ and ∞ the probability necessarily must be 1 (ie. 100%) - so this places a constraint on which functions are valid PDFs.

Example: Normal distribution

The most common PDF is the normal distribution and it'll serve as a frequent example

[1/(σ√2π)] * e([x-Xcenter]2)/(2σ2)

It has two parameters, σ and Xcenter. Other functions may have different parameters. As we will see later, in this case these two happen to correspond to the mean and standard deviation. This is not always the case for all PDFs. 

function [points] = normal_dist(xs, center, sigma)
  normalization_factor = 1/(sigma*sqrt(2*pi))
  points = normalization_factor * e.^(-((xs-center).^2)/(2*sigma^2))
end

We can plot it to see how it looks 

plot_range = 5
sigma = 1
center = 3
xs = [(center-plot_range):(plot_range/50):(center+plot_range)]
plot(xs,normal_dist(xs, center, sigma))
axis "off"
print "-S720,160" "normal.svg"
hold off;

Probability densities

It's important to note that a probability density function f() evaluated at a particular point x0 will not give you the probability of getting the value x0

A truely continious random variable will never generate the same value twice b/c you need to picture each xi as having an infinite number of digits after the decimal. Any specific value x0 has a zero probability of occuring. Any value you've gotten will never reoccur (assuming infinite precision). Therefore the probability of x0 is not a meaingful value nor does f(x0) represent it. None the less, given some measured values x0 x1 x2 .. we still want to be able to have some way to think about how they relate to their associated probability distribution. Saying the probability is just zero for all xi is not particularly useful.

We sidestep the issue by leveraging the definition of the integral of the PDF. The integrating the PDF gives us a probability. If we take a f(xi and multiply it times a very small value dx then it will give us the immediate area under the function at f(xi which will be equivalent to the integral from x_[0}-dx/2 to x_[0}+dx/2 given a small enough dx. Ofcourse as this dx goes to zero this area/probability will also go to zero as well. However as long as the dx is a fixed tiny value then when we compare the tiny probability f(x0)dx wih the tiny probability f(x1)dx and say something useful like which one is likely. In practive b/c all the dx values are the same we can in effect ignore/drop them and just looking at the factors f(x0), f(x1) etc.. to gauge the relative likelihood or different points

Derived values

The unusal thing aboutrandom variables is that we can often skip their probability distributions entirely and summerizing the with just the mean and variance. Sometimes, like in the case of the normal distribution, these values directly match parameters of the underlying distribution (but this isn't necessarily true). What's surprising is that even after we've reduced the underlying probability distributions to a mean and variance we can still, in a limited way, safely manipulate and combine random variables without knowing their distributions at all.

Expected Value

The expected value of a random variable is the "average" of the probability distribution. Usually written as E[x]. For continious functions this translates to the integral ∫f(x)×x from -∞ to ∞

Variance

The variance is a value to describe how much the random variable differs from its expected value

Var[x]= E[(E[x]-x)2]

The power-of-2 ensures that the you are integrating over positive values and it's simply easier than dealing with the absolute value operator. However do bear in mind that squaring gives extra weight to larger deviations. Ofcourse note that if the expected value of the random variable is 0 then the variance is just E[x2] or ∫(f(x))2

Standard deviation

The standard deviation is just the square root of the variance and denoted with σ:

Var[x] = σx2

SigFigs

In many cases, like when taking a measurement with an instrument, we are given values in the form of an expected value plus/minus a standard deviation. The underlying probability distribution will often either be infered from context or irrelevant. When you derive values from these numbers they will also be in the same form.

Uncertainties/standard-deviation should typically be one digit. And they should be rounded to be on the order of the significant figure.

2.45 ± 0.04

In some cases you may want to leave more sigfigs in the uncertainty like if the number is two significant figures and the first digit is small

1.3 ± 0.4
1.3 ± 0.37

You can also express the uncertainty as a a fraction or a percentage. Given an uncertainty

x0 ± δx0

You would write is as

x0 with a [100*δx0/x0] percent uncertainty
or
x0 ± xx%

Adding two random variables

Expected Value

If we add two random variables z=x+y then, regardless of their underlying probability distributions, their estimated values will add directly.

E[z] = E[x+y] = E[x] + E[y]

This is a consequence of integrals of sums ∫(f(x)×x+g(x)×x) = ∫f(x)×x+∫g(x)×x

Variance

The variance will also add directly, irrespective of distribution, as long as x and y are not correlated. To see this, imagine for simplicity that you have two random variables x and y that have an expected value of 0. We will say the random variable z is their sum z=x+y

E[x] = 0
Var(x) = E[0 - x2] = E[x2]
E[y] = 0
Var(y) = E[0 - y2] = E[y2]
E[z] = E[x] + E[y] = 0
Var[z]= E[(E[z]-(z))2] = E[(0-(x+y))2] = E[(x+y)2] = E[x2+y2+xy] = E[x2] + E[y2] + E[xy]

As long as x and y are uncorrelated, we can split the last term E[xy] into E[x]*E[y] to see that it is equal to 0

Standard Deviation

This leave the standard deviation as the "quadrature sum" of the two random variables

σz = √[E[x2] + E[y2]] = √(σx2 + σz2)

Combining the two:

(x±δx) + (y±δy) = q±δq
q = x+y
δq = √{δx2+δy2} 

function [z, dz] = uncertainSum(x, dx, y, dy)
  z = x + y
  dz = sqrt(dx^2+dy^2)
end

[z dz] = uncertainSum(10, 1, 20, 3)
ans = [z dz]

Multiplying two random variables

Expected Value

Working as before.. taking the product of two random variables x and y

E[z] = E[x] × E[y]

Variance

Again we choose x and y to have an expected value of 0. z will be their product z=x×y

We start directly with our equation for the variance

Var(xy) = E[(xy-XY)2]

The inner term we can be factorize into a sum

xy-XY = XY(xy/XY-1)
=XY×( ((x-X)/X+X/X)((y-Y)/Y+Y/Y)-1 )
=XY×( (x-X)×Y/XY + (y-Y)×X/XY + (x-X)(y-Y)/XY)
=XY×( (x-X)×/X + (y-Y)×/Y + (x-X)(y-Y)/XY)

The last term is very small so we drop it

xy-XY ≅ XY×( (x-X)×/X + (y-Y)×/Y

But we need the expected value of the square of this value, so we look at its square

(xy-XY)2 ≅ X2Y2×( (x-X)2/X2 + (y-Y)2/Y2 + 2(x-X)2(y-Y)2/X2Y2 )

The last term is again small and is dropped

(xy-XY)2 ≅ X2Y2×( (x-X)2/X2 + (y-Y)2/Y2 )

So finally the expected value of this (ie. the variance) will be:

Var(xy) = E[(xy-XY)2] ≅ X2Y2× (Var(x)/X2 + Var(y)/Y2)

Standard Deviation

The standard deviation is the square root of the variance..

σxy ≅ √ [X2Y2× Var(x)/X2 + Var(y)/Y2]

but typically you will this written in terms of fractional standard deviations b/c it makes the equation look more symmetric

σxy/XY ≅ √[(σx/X)2 + (σy/Y)2]

Now it's just a straight quatrature sum as before, but instead of standard deviations it's fractional standard deviations

(x±δx) × (y±δy) = q±δq
q = x×y
δq/q = √{(δx/x)2 + (δy/y)2} 

function [z, dz] = uncertainProduct(x, dx, y, dy)
  [x fx] = uncertainty2fractional(x, dx)
  [y fy] = uncertainty2fractional(y, dy)
  z = x*y
  dz = z*sqrt(fx^2+fy^2)
end

function [z, dz] = uncertainDivision(x, dx, y, dy)
  [x fx] = uncertainty2fractional(x, dx)
  [y fy] = uncertainty2fractional(y, dy)
  z = x/y
  dz = z*sqrt(fx^2+fy^2)
end

Estimating the Probability distribution

While we now know how to manipulate and combine independent random variables, in a typically experimental scenario you simply start with a series of measurements. They have some underlying probability distribution f() who's paramters are unknown to us. But there is typically some meta information about the process that generated the measurements which tells you the general formula for the underlying random variable's probability function. Knowing the formula and the measurements we can go as far as writing a function for the cumulative probability of getting all a value in the vicinity of x0 and the x1 and then x2 …

f(x0) dx × f(x1) dx × f(x2) dx × ..

It will have baked in some unknown distribution parameters p1,p2.. - but this will clearly generate very small numbers - and the more measurements you have the smaller they will be. Furthermore as dx gets samller this number will decrease further. As we'll see, the value we choose for dx doesn't actually matter and so we ommit it, but do remember that the unerlying random variable x is continious and the f(xi) values are not a probability of getting a value /xi

f(x0)×f(x1)×f(x2)× ..

The last step is to "guess" those parameters p1,p2.. that maximize this function b/c the best function we can guess is the one that makes these measurements the most likely. This will be our best estimate for the probability distribution. As is most often the case maximization is done by differentiation and setting the result to zero.

Estimating the Normal

As a motivating example, and one that is eminently useful, we will estimate the parameters of the normal distribution given some measurements x0 x1 x2 .. We start off by writing out the measurements' cumulative probability

f(x0) dx × f(x1) dx × f(x2) dx × ..

We know (somehow) that the random variable is normal, so that means:
f(x) = [1/(σ√2π)] e([x-Xcenter]2)/(2σ2)
Plugging it in, we get a cumulative probability will be

[1/(σ√2π)]n e∑ ([xi-Xcenter]2)/(2σ2) dxn

Xcenter and σ are our two unknow parameters

Mean of the Normal

As stated before, the best estimate we can make is the one that will make the cumulative probability highest. It's the distribution that would have had the best chance of generating these measurements.

We now want to find which mean, Xcenter, would maximize the cumulative probability of our measurements. Fortunately here, even though the cumulative probability equation is a bit complicated, we only need to maximize the exponent part. Everything else is not simply not function of Xcenter.

∑ (xi-Xcenter)2/(2σ2)

To find its maximum we differentiating and set equal to zero:

∑ (xi-Xcenter) = 0

Which gives us a

Xcenter = ∑xi/N

An important thing to note at this point is that the final result is luckily completely independent of σ

Standard deviation of the normal

To estimate an optimal standard deviation for the normal distribution we follow the same general pattern as with the mean. We differentiate with respect to the variable we're interested in, σ, and then solve for when it's equal to zero to find the σ that maximizes the likelyhood of generating the measurements

The first complication is that unfortunately we can't just maximize the exponent in this case b/c there is a /sigma in the factor in front as well as in the exponent. So we need to differentiate the whole cumulative probability to find the maximum.

d/dσ [1/(σ√2π)]n e∑ ([xi-Xcenter]2)/(2σ2)

this is a bit messy.. but is done by differentiating by-parts
(Note that I've dropped the dxn factor at the end b/c it's superfluous and has no bearing on maximizing the function)

σbest-estimate = √ (1/N) ∑(xi-Xcenter)2

The next issue we see is that the solution for parameter σ is a function of parameter Xcenter which we also don't know. The best we can do here is to subsitute in the best estimate we'd just found in the previous section: ∑xi/N

This unfortunately introducing a bias! The estimated mean is such that the measurments are situated "optimally" around it. The true mean, whatever it is, will be guaranteed to be worse relative to our measurements and so the measurements in actuality deviate more from the mean than estimated. The true Xcenter of the underlying random variable will not be perfectly centered and the true standard deviation will always be a bit higher. The more correct estimate for σ will be:

σbest-estimate = √ (1/[N-1]) ∑(xi-Xcenter)2

Proof of this is ommitted.. but do note that as N gets larger this bias get insignificant

Uncertain values

To "summarize" a random variable, we will often simply quote it as an expected-value plus/minus a standard deviation (or our best estimates of the two). The underlying probability distribution will often either be infered from context or irrelevant. Most typically the quoted value will be a mean of some normally distributed random variable and the standard deviation will be the standard deviation of the mean (SDOM):

σmean=σ /√N

Additional approximations

Functions of uncertain values

When values go through a function, then the uncertainties of the value lead to uncertainties in the function result. Given a function that takes x to q, ie. q = f(x), and given a meaurement x0 with an uncertainty δx0 - we need to be able to propagate the uncertainty so that we can give the resulting q0 a δq0.

The trick is that if the uncertainty is relatively small then we can approximate the uncertainty as locally linear around q0. If we determine the slope at q0 then we can see how a small change δx0 would change the output q0. Once we have this slope it's just a matter of multiplying the δx by it to get the resulting small change would in q0 - which is our δq0.

To get the local slope we take the derivative of our function

δq=(∂f(x) / ∂x ) δx

Powers of uncertain values

Given an equation xn we would like to know the uncertainty in the output for some input point x0 which has an uncertainty of its own of δx0.

Using the previous section, we take the derivative and we know that the slope of the function is nx0n-1 at x0. So we can say that uncertainty in the output is nx0n-1δx0.

However, observe that the fractional uncertainty that comes out is much simpler. We dividing the output uncertainty by the output value and get the simple equation

[nx0n-1δx0] / [x0n]

=

n[δx0/x0] 

function [z, dz] = uncertainPower(x, dx, n)
  [x fx] = uncertainty2fractional(x, dx)
  z = x^n
  dz = z*n*fx
end

Combining general uncertain values

Differentiation of a function with respect to a variable gives us the slope at every point. A slope tell us how severly any small change in that variable at any given point will affects the output.

In our case the uncertainty of a variable represents the range the value can "change".

Writing it out mathematically, given a a function q that takes an input variable x, the partial-differentiation (∂q / ∂x) gives us this slope for all x, ie the rate of change of q for every at every value of x. We then evaluate it at a given measurement point x0 and this give us an instantaneous slop which represents the sensitivity of the function to change at the point. Once we multiply this slope/sensitivity times that's point's uncertainty we get out how much the output will be uncertain.

When we have multiple variables affecting the output then we repeat these calculations and add them in quadriture.

δq = √[(∂q/∂x)δx)2 + … + (∂q/∂z)δz)2]

Each term is the uncertainty of that one variable's effect on the output of the overall function

Counting experiments

If we measure the occurance of an event N times, then the uncertainty is √N

Mean and standard deviation

Mean

[1/N] ∑ xi

Standard Deviation

Sample standard deviation

σx=√ [1/(N-1)] ∑ (xi-xmean)2

Population standard deviation

σx=√ [1/N] ∑ (xi-xmean)2

These values can be used as an uncertainty δx - and should represent a 68% confidence interval

Standard deviation of the mean SDOM

σmean = σx/[√N]

This represents the best estimate for x

Systematic error

Can not be reduced by additional measurements (denominator in SDOM)

δxtotal=√(δxrandom)2+(δxsystematic)2
